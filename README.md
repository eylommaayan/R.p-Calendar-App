# 📅 Smart React Calendar App

אפליקציית ניהול זמן מודרנית שנבנתה ב-**React**, המשלבת עיצוב עתידני (Dark Mode) עם לוגיקה מורכבת של ניהול תאריכים. הפרויקט התחיל כמימוש של עקרונות ה-React ומתפתח לכלי חכם המבוסס על **AI**.

## 🚀 פיצ'רים מרכזיים
* **ממשק לוח שנה דינמי:** חישוב תאריכים, חודשים וימי שבוע בזמן אמת (Vanilla JS Logic).
* **ניהול אירועים (CRUD):** הוספה, עריכה, מחיקה וסידור אוטומטי של אירועים לפי שעה.
* **עיצוב Ultra-Modern:** שימוש ב-CSS מתקדם, Glassmorphism ורספונסיביות מלאה למובייל.
* **תיקוף נתונים:** הגבלת תווים ובדיקת תקינות זמנים.



## 🛠️ טכנולוגיות
* **Frontend:** React.js (Hooks, Functional Components)
* **Styling:** CSS3 (Flexbox, Grid, Media Queries)
* **Icons:** Boxicons

---

## 🤖 חזון ה-AI (Roadmap)
הפרויקט נמצא בתהליך שדרוג להפיכתו ל"יומן חכם" בעזרת בינה מלאכותית:
1. **Natural Language Input:** הוספת אירועים בכתיבת משפט חופשי (למשל: "פגישה בשישי בצהריים").
2. **Smart Categorization:** זיהוי אוטומטי של סוג האירוע והצמדת אייקון וצבע מתאימים.
3. **Daily Summary:** סיכום יומי חכם ותעדוף משימות מבוסס AI.

---

## 🎨 דגשים בעיצוב (UI/UX)
* **Glassmorphism:** שימוש בטשטוש רקע (Blur) ושקיפות ליצירת עומק.
* **Interactive Elements:** אנימציות מעבר חלקות ב-Hover ולחיצה.
* **Accessibility:** היררכיה ברורה ופונטים קריאים מותאמי Web.
